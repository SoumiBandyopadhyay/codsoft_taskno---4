# 🎮 Rock Paper Scissors Game

## 📌 Project Description
This is a simple **Rock Paper Scissors Game** developed in Python.  
The user plays against the computer, which randomly selects rock, paper, or scissors.

The game determines the winner based on standard game rules and keeps track of scores across multiple rounds.

---

## ✨ Features
- User-friendly interface
- Random computer choice
- Game result display
- Score tracking system
- Multiple rounds support
- Input validation
- Play again option

---

## 🕹️ Game Rules
- 🪨 Rock beats Scissors
- ✂️ Scissors beats Paper
- 📄 Paper beats Rock

---

## 🛠️ Technologies Used
- Python
- Random Module
